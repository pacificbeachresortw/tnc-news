API 金鑰設定
GitHub Token 設定（自動發布文章頁必填）
  1. 前往 github.com/settings/tokens
  2. 點 Generate new token (classic)
  3. Note 填入 TNC News Admin
  4. Expiration 選 No expiration（或自訂）
  5. 勾選 repo（完整 repo 存取）
  6. 點 Generate token，複製 token（ghp_...）

填入欄位：

  • GitHub Token：貼上剛複製的 ghp_xxx
  • GitHub Repo：填入 pacificbeachresortw/tnc-news
  • 網站網址：填入 https://taiwannc.com

設定完成後，每次儲存新聞時系統會自動在 GitHub 建立 pages/news/{id}.html 檔案。

JSONbin Master Key
  1. 前往 jsonbin.io 註冊帳號
  2. 登入後點右上角頭像 → API Keys
  3. 複製 Secret Key（以 $2a$ 開頭）
  4. 貼入左側「JSONbin Master Key」欄位
JSONbin Bin ID
  1. 登入 jsonbin.io 後點左側 Bins
  2. 點 + Create Bin 新增一個 Bin
  3. 初始內容輸入：{"news":[]}
  4. 建立後複製網址列中的 Bin ID（24位英數字串）
  5. 貼入「JSONbin Bin ID」欄位
ImgBB API Key
  1. 前往 imgbb.com 註冊帳號
  2. 登入後前往 api.imgbb.com
  3. 點 Get API key
  4. 複製 API Key 並貼入「ImgBB API Key」欄位
  5. 此 Key 用於上傳封面圖片
API 金鑰儲存於瀏覽器的 localStorage，不會傳送至任何第三方伺服器。建議在私人裝置上使用後台。設定完成後點「儲存設定」，再點「測試連線」確認是否正常運作。
